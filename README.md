# Die-To-Win-By-Pinegrape1213
Die To Win — A Platformer Where Death Is the Key

Die To Win is a fast-paced 2D platformer built with C++ and SFML, featuring 14 handcrafted levels, smooth controls, and a clever twist: sometimes you must die to progress. With clean visuals, responsive physics, and progression saving, the game focuses on skill, timing, and creative level design.
